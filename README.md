> Uma experiência de gestão financeira cinematográfica com estética **Deep Space**.

**Finance Obsidian** é um dashboard financeiro de alta performance projetado para oferecer clareza absoluta sobre suas finanças pessoais. Combinando o poder do **Supabase** para dados em tempo real e uma interface inspirada na profundidade do cosmos.

---

## 🌌 Visão Geral

O projeto foi construído com foco em **Visual Excellence** e **Actionable Insights**. Através de uma interface "Glassmorphism" escura com acentos neon, o usuário pode gerenciar transações, monitorar contas a vencer e visualizar projeções de saldo futuro com precisão matemática.

### 🚀 Funcionalidades Principais

- **Dashboard Holístico**: Resumo de saldo total, entradas e saídas mensais com indicadores de saúde financeira.
- **Análise de Fluxo Dinâmica**: Gráficos de linha interativos que mostram a evolução do saldo dia a dia.
- **Gestão de Contas a Vencer**: Sistema de agendamento com alertas visuais para contas pendentes, pagas ou atrasadas.
- **Projeção de Saldo (Fase 7)**: Cálculo inteligente de saldo futuro baseado em receitas fixas e contas agendadas (horizontes de 7 a 60 dias).
- **Central de Alertas (Fase 8)**: Notificações inteligentes sobre o status das suas contas.
- **Segurança com Supabase**: Autenticação robusta e banco de dados relacional seguro.

## 🛠️ Tecnologias

- **Core**: HTML5, Vanilla CSS3 (Custom Design System)
- **Backend/Auth**: [Supabase](https://supabase.com/)
- **Gráficos**: [Chart.js](https://www.chartjs.org/)
- **Ícones**: [Lucide Icons](https://lucide.dev/)
- **Tipografia**: Outfit & Inter (Google Fonts)

---

## 🎨 Design System: Deep Space

O dashboard utiliza uma paleta de cores inspirada em nebulosas e no espaço profundo:
- **Solar Cyan** (`#00F2FF`): Ações primárias e destaques de sucesso.
- **Nebula Purple** (`#8A2BE2`): Acentos secundários e fluxos de dados.
- **Pulsar Pink** (`#FF00E5`): Alertas críticos.
- **Deep Background** (`#010409`): Base de alto contraste para redução de fadiga visual.

## ⚙️ Configuração Local

1. Clone o repositório:
   ```bash
   git clone https://github.com/JeanVictor15/finance-obsidian-premium.git
   ```

2. Configure suas chaves do Supabase em `dashboard/script.js` (ou use variáveis de ambiente em um build step futuro).

3. Abra o `index.html` em um servidor local (Live Server recomendado).
